## **权限添加**

如果用户仅使用SpotMax SaaS提供的资源使用分析功能，则需要添加以下权限

| 权限名称 | 权限说明 |
| :--- | :--- |
| AmazonEC2ReadOnlyAccess | Provides read only access to Amazon EC2 via the AWS Management Console. |
